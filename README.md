# Light House — Printing House Management System

A web-based management system for a printing house to handle customers, orders, production workflows, invoices, and staff.

## Tech Stack

| Layer | Technology |
|---|---|
| Backend | Laravel 13 / PHP 8.4 |
| Frontend | Vue 3 SPA + Tailwind CSS 4 |
| Bundler | Vite 8 |
| Database | MySQL 8.0 |
| Auth | Laravel Sanctum (Bearer token) |
| Permissions | Spatie Laravel Permission |
| Runtime | Docker (PHP-Apache + Node + MySQL) |
| Local email | Mailcatcher |
